# recipes
This module provide some modules recipe for build kivy app to ios and android OS
there is a list to show if the recipes passes test or not

there is a android folder under the recipes folder, all the android modules's recipe stores inside android folder

there is alse a ios folder under the recipes folder, all ios modules's recipe are store inside ios folder
## ios

### iospr
[iospr](docs/iospr.md) A recipe generator for module need to install to root/python3

### ioshpr
[ioshpr](docs/iospr.md) A recipe generator for module need to install to hostpython3

### iosln
link all the recipes and source code to site-package/kivy_ios/recipes
