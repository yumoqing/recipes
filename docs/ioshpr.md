# ioshpr 
A recipe generator accountly by pattern, for module will install to hostpython3

## Usage
ioshpr module_name[==version] [recipe_folder_name]

it will generator a recipe folder with a __init__.py in it

if recipe_folder_name is omited, module_name will be using as recipe filder name.

when the module_name contains '-', must use the recipe_folder_name.
